README

Run the appropriate script to execute the jar; make sure you have (at least) the JRE installed and the java runtime in your path.

	.bat for Windows
	.command for OSX
	.sh for Unix/Linux

Capitalization of elements matters (CO is carbon monoxide, Co is cobalt) -- append subscripts as numbers following the relevant element (CO2 is carbon dioxide).
Separate reactants/products with "+" (spaces are optional) and the reactants and products with a "=".

Made by Jason Guo, last updated 9/4/2016.
